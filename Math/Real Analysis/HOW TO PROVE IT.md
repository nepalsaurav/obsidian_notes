# Unit 1. Sentential Logic

- **Deductive Validity:** An argument is **valid** if the conclusion must be true whenever the premises are true. Validity depends on **logical form**, not the topic.
    
- **Logical Connectives:**
    
    - **$\lor$ (Disjunction):** "Or"
        
    - **$\land$ (Conjunction):** "And" (or "but")
        
    - **$\neg$ (Negation):** "Not"
        
- **Well-Formed Formulas:** Symbols must combine statements into clear, grammatical structures (e.g., $P \land \neg Q$).
    
- **Syntax Rules:**
    
    - $\neg$ applies to the immediate next statement.
        
    - Parentheses group operations to avoid ambiguity.
        
- **Mathematical Context:** Notation often hides logic.
    
    - $x \le y$ means $(x < y) \lor (x = y)$.
        
    - $a < b < c$ means $(a < b) \land (b < c)$.
